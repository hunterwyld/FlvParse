## FlvParse

FlvParse is a desktop app which shows Flv file structures.

### Download
For macOS, an executable is available [here]()

For other OS:
```
git clone https://github.com/hunterwyld/FlvParse.git
cd FlvParse
mvn clean package
java -jar target/flvparse.jar
```
